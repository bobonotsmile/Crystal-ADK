# crystal-adk v0.1.0 架构设计稿

## 文件树

```text
crystal-adk/
├─ Core/
│  ├─ AgentContracts.cs        // 统一运行时消息、工具调用、运行输入输出结果模型
│  ├─ AgentHost.cs             // 运行编排中心；非流式、流式、tool loop、事件发射都在这里
│  ├─ AgentHostOptions.cs      // Host 默认配置；system prompt 与 max rounds
│  ├─ Events.cs                // 统一运行事件类型与事件载体
│  ├─ Providers.cs             // Provider 抽象接口与统一流式分片模型
│  └─ ToolHub.cs               // 工具注册、工具描述收集、工具执行
├─ Providers/
│  ├─ ChatProvider.cs          // Provider 工厂入口；按 vendor 创建具体实现
│  ├─ ChatProviderSettings.cs  // 对外统一 provider 配置模型
│  ├─ ProviderJson.cs          // tool arguments 的 JSON 归一化工具
│  ├─ ProviderOptions.cs       // provider 内部专用 options 模型
│  ├─ Ark/
│  │  └─ ArkChatProvider.cs    // ARK 协议适配、HTTP 发送、响应解析
│  └─ Ollama/
│     └─ OllamaChatProvider.cs // Ollama 协议适配、HTTP 发送、响应解析
└─ Session/
   ├─ AgentSession.cs          // 会话入口；保存消息历史；包装 Host
   └─ TextChunk.cs             // 对外文本流分片模型
```

## 文件关系

```text
Session/AgentSession.cs
  -> Core/AgentHost.cs
  -> Core/AgentContracts.cs
  -> Session/TextChunk.cs

Core/AgentHost.cs
  -> Core/AgentContracts.cs

  -> Core/AgentHostOptions.cs

Providers/ChatProvider.cs

  -> Providers/Ark/ArkChatProvider.cs

Providers/Ark/ArkChatProvider.cs
  -> Core/AgentContracts.cs
  -> Core/Providers.cs
  -> Providers/ProviderJson.cs
  -> Providers/ProviderOptions.cs

Providers/Ollama/OllamaChatProvider.cs
  -> Core/AgentContracts.cs
  -> Core/Providers.cs
  -> Providers/ProviderJson.cs
  -> Providers/ProviderOptions.cs
```

## 总体主链路

```text
业务代码
-> Session/AgentSession.cs
-> Core/AgentHost.cs
-> Core/Providers.cs : IChatProvider
-> Providers/Ark/ArkChatProvider.cs 或 Providers/Ollama/OllamaChatProvider.cs
-> 厂商 HTTP API
-> Core/AgentContracts.cs : RuntimeMessage / AgentRunResult
-> Session/AgentSession.cs 更新消息历史
```

## 局部主链路

### 非流式

```text
AgentSession.RunAsync
-> AgentHost.RunAsync
-> IChatProvider.CreateCompletionAsync
-> RuntimeMessage assistant
-> AgentHost tool loop 或 final answer
-> AgentRunResult
```

### 文本流

```text
AgentSession.StreamTextAsync
-> AgentHost.RunStreamingAsync
-> IChatProvider.StreamCompletionAsync
-> StreamingChatChunk
-> AgentEventKinds.TextDelta
-> TextChunk
```

### 事件流

```text
AgentSession.StreamEventsAsync
-> AgentHost.RunAsync(onEvent)
-> AgentEvent
```
