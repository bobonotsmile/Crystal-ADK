# crystal-adk v0.2.0 数据结构稿

## `Abstractions/RuntimeMessage.cs`

```csharp
public sealed class RuntimeMessage
{
    // 消息角色：system / user / assistant
    public string Role { get; set; } = "user";

    // 消息正文内容
    public string? Content { get; set; }

    // 消息附带的 thinking / reasoning 文本
    public string? ThinkingContent { get; set; }
}
```

## `Abstractions/TextChunk.cs`

```csharp
public sealed class TextChunk
{
    // session 对外暴露的正文流分片
    public string Text { get; init; } = string.Empty;

    // session 对外暴露的 thinking 流分片
    public string ThinkingText { get; init; } = string.Empty;
}
```

## `Abstractions/StreamingChatChunk.cs`

```csharp
public sealed class StreamingChatChunk
{
    // provider 产出的正文增量
    public string? TextDelta { get; init; }

    // provider 产出的 thinking 增量
    public string? ThinkingDelta { get; init; }

    // 本轮流式输出是否结束
    public bool IsCompleted { get; init; }
}
```

## `Providers/ChatProviderOptions.cs`

```csharp
public sealed class ChatProviderOptions
{
    // 厂商名：ark / ollama
    public string Vendor { get; init; } = "ollama";

    // 认证信息
    public string ApiKey { get; init; } = string.Empty;

    // 模型名
    public string Model { get; init; } = string.Empty;

    // 请求地址
    public string BaseUrl { get; init; } = string.Empty;

    // 超时
    public int TimeoutMs { get; init; } = 30000;

    // 温度
    public double? Temperature { get; init; }

    // top_p
    public double? TopP { get; init; }

    // 输出 token 上限
    public int? MaxOutputTokens { get; init; }

    // thinking 开关；为空表示不主动表达该配置
    public bool? EnableThinking { get; init; }
}
```

## `Session/SessionMessageManager.cs`

```csharp
public sealed class SessionMessageManager
{
    // 会话消息内部存储
    private readonly List<RuntimeMessage> _messages = new();

    // 当前消息只读视图
    public IReadOnlyList<RuntimeMessage> Messages => _messages;
}
```

说明：

```text
内部状态：
  _messages

对外暴露：
  Messages

外部主要操作对象：
  RuntimeMessage

assistant 历史消息当前可同时保存：
  Content
  ThinkingContent
```

## `Session/AgentSession.cs`

```csharp
public sealed class AgentSession
{
    // provider 抽象
    private readonly IChatProvider _provider;

    // 历史消息操作入口
    private readonly SessionMessageManager _messageManager;

    // 历史操作入口对外暴露
    public SessionMessageManager History => _messageManager;

    // 当前消息历史只读视图
    public IReadOnlyList<RuntimeMessage> Messages => _messageManager.Messages;
}
```

说明：

```text
AgentSession 自身不额外保存消息字段
消息状态统一落在 SessionMessageManager
流式阶段会分别累积：
  fullText
  fullThinking
```

## `Providers/Interfaces/IChatProvider.cs`

```csharp
public interface IChatProvider
{
    // 非流式完整 assistant message
    Task<RuntimeMessage> CompleteAsync(...);

    // 流式统一分片
    IAsyncEnumerable<StreamingChatChunk> StreamAsync(...);
}
```

说明：

```text
IChatProvider 本身不承载数据字段
它定义的是 provider 的外部能力边界
返回结构当前可承载：
  正文
  thinking
```

## `Providers/Models/Ark/ArkChatProvider.cs`

```csharp
internal sealed class ArkChatProvider : IChatProvider
{
    // 共享 HttpClient
    private readonly HttpClient _httpClient;

    // provider 配置
    private readonly ChatProviderOptions _options;
}
```

说明：

```text
ArkChatProvider 的长期状态字段只有：
  _httpClient
  _options

请求体、响应文本、流读取器都在函数内部短生命周期存在
assistant 返回中的 reasoning_content 会被映射到：
  RuntimeMessage.ThinkingContent
  StreamingChatChunk.ThinkingDelta
```
